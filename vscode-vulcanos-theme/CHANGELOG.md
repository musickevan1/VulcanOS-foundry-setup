# Changelog

All notable changes to the "VulcanOS Theme" extension will be documented in this file.

## [1.0.0] - 2026-02-01

### Added
- Initial release of VulcanOS Dark theme
- Complete color definitions for editor, UI, and terminal
- Syntax highlighting with warm forge-inspired palette
- Semantic token support for modern LSPs
- Terminal ANSI colors matching VulcanOS system palette
- Git decoration colors
- Comprehensive UI element theming (sidebar, panels, tabs, etc.)
