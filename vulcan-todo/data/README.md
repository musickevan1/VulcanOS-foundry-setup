# Vulcan-Todo Data Directory

This directory is used by vulcan-todo for storing task data and related files.

## Files

- `tasks.json` - Main task storage file

## Directory Structure

```
~/.config/vulcan-todo/
├── tasks.json    # Task database
└── logs/         # Application logs (optional)
```

## Backup

Simply back up `tasks.json` to version control your tasks!
